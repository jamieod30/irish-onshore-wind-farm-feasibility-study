# irish-onshore-wind-farm-feasibility-study
Wind resource assessment and feasibility analysis of a grid-connected onshore wind farm in Ireland, including layout optimisation, energy yield modelling, and storage integration analysis.

This repository presents a preliminary feasibility study for an onshore wind farm in Ireland. The project follows the workflow typically used in renewable energy development, including regional wind resource screening, site selection, wind farm layout optimisation using WindPro, energy yield assessment, and analysis of grid integration and energy storage scenarios.

The study combines geospatial analysis, wind resource modelling, and techno-economic evaluation to explore how wind energy projects can be optimally developed within the Irish electricity system.
